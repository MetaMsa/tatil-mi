# Kar Tatili Projesi ❄️

**Kar Tatili Projesi**, Türkiye’deki güncel kar tatili duyurularını takip edebilmenizi sağlayan bir web uygulamasıdır.  

## Özellikler

- Güncel kar tatili haberlerini tek bir yerden görüntüleme  
- Kaynak ve metodoloji bilgileri ile şeffaflık  
- Hatalı veya eksik bilgilere karşı uyarılar  
- Kullanıcı dostu ve hızlı arayüz  

## Teknolojiler

- **Frontend:** Next.js, React  
- **Backend & Database:** Prisma, PostgreSQL  
- **Diğer:** Vercel (yayın), TailwindCSS (tasarım)  

## Kurulum

1. Repository’yi klonlayın:
```bash
git clone https://github.com/kullanici/kar-tatili.git
```

2. Gerekli paketleri yükleyin:
```bash
npm install
```

3. Ortam değişkenlerini ayarlayın:
```bash
cp .env.example .env
```

4. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

Projeye tarayıcınızdan http://localhost:3000 üzerinden erişebilirsiniz.

## Kullanım

Projeyi ziyaret ederek güncel kar tatili haberlerini görüntüleyebilir, metodoloji kısmını inceleyebilir ve geri bildirim bırakabilirsiniz.
